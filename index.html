How to use: Place characters in sequence separated by spaces.
Non-separated characters will be played concurrently. ' k ' : Kick
' s ' : Snare
' h ' : Hi-Hat
' x ' : Rest Example: " kh h ksh h "
Video Demo: https://youtu.be/EiNw-OAn5N8
Blog Post: This project is a small drum kit made using WebAudio with a kick, snare, and hi-hat.
Each drum has a list of sliders to customize its parameters, in addition to play buttons for previewing. This includes volume, attack, and decay for each drum. I created the kick using a sine wave that starts at a specified frequency and falls to another over an interval specified by decay. The user can set both start and end frequencies. My snare consists of white noise with a highpass filter and a triangle wave. The filter and oscillator frequencies can be set, as well as the gain and decay ratios of the oscillator relative to the noise. The hi-hat is just white noise with a band pass filter, again with customizable frequency and q value. The project includes a basic sequencer, using characters for each drum. Characters are placed in sequence separated by spaces, where non-separated characters are played concurrently. The sequencer includes a BPM setting as well as a rest character. Drum parameters can be changed while the sequencer is playing, allowing the user to hear their changes in realtime.
